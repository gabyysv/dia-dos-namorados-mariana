# Eu te amo muito

Pagina romantica para publicar no GitHub Pages.

## Como personalizar

Depois de publicar, use os parametros na URL:

```text
https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/?nome=Maria&foto=https://exemplo.com/foto.jpg
```

- `nome`: aparece na ultima pagina da frase "eu te amo muito".
- `foto`: imagem usada como fundo de cada pagina.
- `mensagem`: opcional, troca as frases abaixo dos slides.

Exemplo:

```text
index.html?nome=Maria&foto=https://exemplo.com/foto.jpg
```
